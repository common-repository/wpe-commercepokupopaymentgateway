Платежный модуль Pokupo для WordPress+eCommerce

Тестировалось на WordPress 4.9.4 + eCommerce 3.14.0

1. В админке Wordpress перейдите в `Плагины->Добавить новый`, нажмите кнопку `Загрузить плагин`
2. Выберите скачанный архив с модулем и нажмите `Установить`
2.1 В случае проблем с установкой скопируйте содержимое архива в папку вашего сайта `wp-content/plugins`
3. Перейдите в `Плагины->Установленные`, найдите `WP e-Commerce Pokupo Payment Gateway` и нажмите `Активировать`
4. Перейдите в `Настройки->Магазин->Платежи`
5. Выберите Pokupo и нажмите `Сохранить изменения`
6. Перейдите в настройки Pokupo
7. В поле `ID магазина` введите 'Идентификатор магазина для CMS' из Настроек магазина в личном кабинете Покупо
8. В поле `Пароль уведомлений ` введите 'Пароль уведомления' из Настроек магазина в личном кабинете Покупо
9. В Настройках магазина в личном кабинете Покупо в поле URL уведомлений укажите адрес из строки `URL уведомлений`
